# 比較透過 Analytics 來源連接器傳遞的 Analytics 資料的術語

Adobe Analytics、資料摘要、Analytics來源聯結器/資料湖和Customer Journey Analytics之間存在一些術語差異。 此主題旨在強調並釐清這些差異。
