# 專案概觀

專案是分析工作區中使用的首要「檔案」。 它們包含一或多個面板，可讓您組織視覺化。

使用專案管理員來 [管理](manage.md)[或建](create.md) 立Workspace專案。 如果您想要進一步瞭解如何在分析工作區中使用專案，請參閱工 [作區基本資訊](../../projects/workspace-basics.md)。
