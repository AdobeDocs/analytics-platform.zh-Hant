# 建立 專案

在分析工作區中 [建立專案](/help/analysis-workspace/home.md) ，有兩種方法：

* 按一 [!UICONTROL 下頁首中] 的「專案」，然後按一 [!UICONTROL 下「建立新專案」]。
* 按一 [!UICONTROL 下「元件] > [!UICONTROL 專案]」，然後按一 ![下「新增](../assets/add.png)專案」。

建立專案時會顯示空白的工作區專案：

![空白專案](../assets/blank-project.png)
