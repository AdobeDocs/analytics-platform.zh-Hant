# 建立 專案

在分析工作區中建立專案有兩種方法：

* 按一 [!UICONTROL Projects] 下頁首中的，然後按一下 [!UICONTROL Create New Project]。
* 按一 [!UICONTROL Components] 下> [!UICONTROL Projects]，然後按一 ![下新增](../assets/add.png)[!UICONTROL Add]。

建立專案時會顯示空白的工作區專案：

![空白專案](../assets/blank-project.png)
