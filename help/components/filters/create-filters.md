# 建立篩選

「篩選產生器」提供畫布來拖放量度、維度、篩選器和事件，以根據容器階層邏輯、規則和運算子來篩選訪客。 此整合開發工具可讓您建立並儲存簡單或複雜的篩選，以識別跨瀏覽和頁面點擊的訪客屬性和動作。

若要存取「客戶歷程分析」中的「篩選產生器」，請按一 **[!UICONTROL Components > Filters]**&#x200B;下，然後按一下。
