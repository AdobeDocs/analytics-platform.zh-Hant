# 日期範圍總覽

日期範圍是您可以在 Workspace 專案中使用的元件類型。它們讓您能看到資料的長期趨勢或事件最常發生的時間。日期範圍以紫色編碼。

自訂日期範圍可讓您自訂在 Workspace 專案中看到的日期。使用日期範圍管理器來[管理](manage.md)或[建立](create.md)自訂日期範圍。
