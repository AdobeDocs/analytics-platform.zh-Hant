# 尋找量度

您可以在Analysis Workspace中找出量度，其方式與找出其他型別元件的方式相同。

如需有關如何快速找到元件的資訊，請參閱[元件概觀](/help/components/overview.md#manage-the-component-list)中的[管理元件清單](/help/components/overview.md)。
