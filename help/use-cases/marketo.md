# 將B2B資料導入AEP並在CJA中報告

利用AEP上新提供的B2B資料集，為B2B營銷人員提供有價值的分析和報告解決方案。 添加儀表板，使客戶能夠直觀地看到通過AEP流動的B2B資料，以便在AEP使用情形中為B2B供電的資料中增加透明度。
