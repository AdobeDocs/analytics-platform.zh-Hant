# 區域圖和堆疊區域圖

## 區域 {#Area}

此視覺效果類似折線圖，但折線下方有一塊上色區域。有多個量度且以要視覺化方式表示多個量度之間交會的區域時，可使用區域圖。

![](assets/area.png)

## 堆疊區域圖 {#stacked}

此視覺效果類似區域圖，但各數列始於前一個數列的頂端。

![](assets/area-stacked.png)

針對堆疊區域圖的全新視覺效果設定可讓圖表以「100% 堆疊」的視覺效果方式呈現。

![](assets/areastacked100.png)
