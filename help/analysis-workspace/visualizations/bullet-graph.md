# 項目符號圖表

可讓您查看您感興趣的值與其他效能範圍 (目標) 相比較或進行測量的結果。

![](assets/bullet-image.png)

項目符號圖表中含有單一的主要測量值 (例如今年至今的收入)，且可用於輸入質化的成效範圍 (例如與目標收入比較)。您可以指定「高」、「中」和「低」的範圍。您可在「[!UICONTROL 視覺效果設定]」中指定目標範圍。
