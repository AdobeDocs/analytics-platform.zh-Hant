# 空白面板

空白面板會反白標示您可以從面板中建立的所有視覺效果，包括「自由形式」、「地圖」、「流失」、「流量」、「長條圖」、「同類群組」和「Venn 圖表」。您也可以選擇將「空白面板」設為預設的專案開始狀態，而非自由形式面板。
