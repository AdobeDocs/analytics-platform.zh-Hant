# 「媒體播放時間」面板

您可以分析所花費的播放時間，以了解發生高峰期同時觀看人數或客戶流失的時間，以針對內容品質和檢視者參與度提供重要解析，並有助於疑難排解或完成數量和規模的相關規劃。

在 Analysis Workspace 中，「播放時間」是指在特定時間觀看您的媒體串流所花費的時間，包括暫停、緩衝和開始時間。

「媒體播放時間」面板可進行一段時間的播放分析，提供高峰期同時觀看人數的詳細資訊，並具備劃分和比較的能力。若要存取「媒體播放時間」面板，可瀏覽至啟用Media Analytics元件的資料檢視。 然後，按一下最左側的面板圖示，並將面板拖放到您的 Analysis Workspace 專案中。

此面板也包括行事曆中的新功能，讓您能選取和顯示 24 小時以內的活動。您可以對整個面板這麼做，或者您可以使用連續時段來建立篩選器，以便跨方案或方案區段追蹤收視率的引進/引出。 當您至少放置其中的兩個日期篩選條件後，您會看到「日期序列顯示」的單選按鈕選項，該選項會覆蓋具有常見x軸起點的線條，或依其特定x軸起點的順序顯示線條。
