# 對象分析概觀

<!-- add hidden text in this article when this feature releases: /help/components/audiences/audiences-overview.md -->

您可以將Experience Platform設定檔資料集中的對象會籍資料擷取到Customer Journey Analytics連線。 (如需有關建立在Customer Journey Analytics中發現的對象，並將其發佈到Adobe Experience Platform以用於客戶目標定位和個人化的資訊，請參閱[對象發佈總覽](/help/components/audiences/audiences-overview.md))

有了Customer Journey Analytics提供的受眾資料，您就可以針對受眾成員在不同管道中的行為方式獲得可行的深入分析。

例如，您可以追蹤包含在相同電子郵件促銷活動中的個別客戶的行為。 有了Customer Journey Analytics提供的對象，您可以看到有關每個對象成員的下列資訊：

* 從電子郵件點進至網站，最後導致購買

* 最終進行店內購買的受眾成員
